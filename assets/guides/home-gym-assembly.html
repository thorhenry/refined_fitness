Home Gym Assembly Guide

Professional Multi-Station Home Gym

Required Tools

  • Adjustable wrench
  • Allen key set
  • Phillips screwdriver
  • Rubber mallet
  • Level
  • Measuring tape

⚠️ Safety First

  • Assembly requires 2-3 people
  • Clear a space of at least 10' x 10'
  • Keep children and pets away during assembly
  • Wear appropriate safety gear

Assembly Steps

Step 1: Frame Assembly

  1. Lay out all components and verify against parts list
  2. Assemble main support frame
  3. Attach stabilizer bars
  4. Level the frame using adjustable feet

Step 2: Weight Stack Installation

  1. Install guide rods
  2. Lubricate guide rods
  3. Stack weight plates in correct order
  4. Install selector pin and test movement

Step 3: Cable System

  1. Install all pulleys
  2. Route cables according to diagram
  3. Verify proper cable tension
  4. Test smooth operation

Step 4: Stations Assembly

  1. Attach press arms
  2. Install leg developer
  3. Mount lat pulldown bar
  4. Secure all padding and seats

Final Checks

  • Verify all bolts are tight
  • Check cable tension at all points
  • Test each station with light weight
  • Ensure smooth operation of all moving parts

Important Notes

  • Maximum weight capacity: 400lbs
  • Required ceiling height: 7'2" minimum
  • Floor space needed: 8' x 6'
